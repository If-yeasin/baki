# Baki Support

For beta and App Store preparation, use this support contact:

- Email: support@baki.app

## Support Scope

- OTP sign-in issues.
- Khata invite, membership, and ledger issues.
- Offline sync repair questions.
- bKash/Nagad settlement handoff questions.
- Account deletion requests or questions.

Do not send OTP codes, JWTs, full bKash/Nagad numbers, or payment transaction references in support screenshots or logs.
