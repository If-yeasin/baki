Delete in the App

  1. Open Baki and sign in.
  2. Go to Settings.
  3. Open Delete account.
  4. Review the warning and confirm deletion.

What Gets Deleted

Deletion removes the user's profile, membership rows, push tokens, and saved payment handoff details. Shared ledger rows remain for other group members, with the deleted user replaced by a deleted-user label.

When Deletion Is Blocked

Baki blocks account deletion while the user has unsettled balances in active groups. Settle those balances first, then repeat the deletion flow.
